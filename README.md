# Analyzing Customer Churn in Excel

This case study was completed through DataCamp.

### Content
- `analyzing-customer-churn.xlsx` is workbook on Excel
- `certificate.pdf` is the certificate of completion for this project

### Abstract
Using a fictitious datset, I implemented my Excel skills to analyze churn rates from example telecom company Databel. I conducted exploratory data analysis to investigate reasons for customer churn and created PivotTables and visualizations to summarize my findings.
